  Interrogar la tecnología para codiseñar la enseñanza
        
Iniciar un taller de codiseño en la enseñanza universitaria requiere, ante todo, revisar nuestras certezas sobre aquello con lo que diseñamos. ¿Qué entendemos por “tecnología educativa”? ¿Cómo se configura su presencia en nuestras prácticas cotidianas? ¿Qué relaciones establece con el conocimiento, el poder, los cuerpos y los afectos?
Esta primera unidad propone una apertura epistemológica: interrogar la tecnología en la contemporaneidad implica asumir que no se trata solo de una herramienta funcional, sino de una mediación cultural, cognitiva, afectiva y política que participa activamente en la construcción del mundo educativo. Desde esta mirada, enseñar con tecnologías implica también pensar con ellas, como parte de ensambles sociotécnicos donde se entrelazan humanos, no humanos, materiales, simbólicos y afectivos.
El codiseño se presenta aquí como una práctica situada, relacional y no neutral. Una invitación a recuperar el carácter colectivo, afectivo y político de nuestras decisiones tecnopedagógicas, entendiendo que toda configuración educativa es, en sí misma, un acto de diseño y una toma de posición epistemológica.
Como plantea Jane Bennett (2010), “la agencia no es propiedad exclusiva de los humanos, sino que emerge de la interacción de múltiples fuerzas en un ensamblaje”. Desde esta perspectiva, el diseño educativo ya no puede pensarse como un proceso lineal o meramente instrumental, sino como una práctica enredada con plataformas, normativas, algoritmos, textos, cuerpos, políticas y deseos.
Este primer paso no busca resolver, sino abrir preguntas, desnaturalizar lo dado, sospechar de lo evidente. Y, sobre todo, habilitar una mirada crítica y creativa sobre cómo enseñamos y qué mundos producimos cuando diseñamos con tecnologías.
Nos proponemos así compartir un proceso de codiseño en un escenario educativo basado en la indagación, en la construcción de interrogantes genuinos que surgen del encuentro entre tecnologías, contextos y prácticas en nuestros entornos educativos. Una mirada que, como advierte Tripaldi (2023), reconoce que “todo diálogo con la tecnología sucede en un territorio híbrido en el que nuestros instrumentos influyen en nuestro comportamiento tanto como nosotres influimos en ellos.” 

        
Aperturas - Interrogantes: ¿Qué entendemos por tecnología educativa cuando codiseñamos propuestas de enseñanza? ¿Estamos diseñando con tecnologías o también estamos diseñando las formas en que pensamos con ellas?
Eje central: El codiseño como acto epistemológico: abrir preguntas sobre las narrativas dominantes de la tecnología educativa. Pensar la tecnología más allá de su función instrumental, como una forma de pensamiento y existencia.
 “Todo diálogo con la tecnología sucede en un territorio híbrido en el que nuestros instrumentos influyen en nuestro comportamiento tanto como nosotres influimos en ellos” (Tripaldi, 2023)

  const tiles   = [];

  const img = new Image();
  img.src = urlImagen;
  img.onload = () => {
    const aspect = img.naturalWidth / img.naturalHeight;

    for (let i = 0; i < cols*rows; i++) {
      const t = document.createElement('div');
      t.className = 'tile-reveal';
      grid.appendChild(t);
      tiles.push(t);
    }

    const total = tiles.length;
    const order = Array.from({length: total}, (_,i)=>i);
    for (let i = total-1; i>0; i--) {
      const j = Math.floor(Math.random()*(i+1));
      [order[i], order[j]] = [order[j], order[i]];
    }
    const revealRank = [];
    order.forEach((idx, rank)=>{ revealRank[idx]=rank; });

    let startY, revealEndY, holdEndY;

    function resizeAll(){
      const vh = window.innerHeight;
      const colWidth = wrapper.parentElement.clientWidth;
      const vwRaw = vh * aspect;
      const vw = Math.min(vwRaw, colWidth);
      const tileW = vw/cols, tileH = vh/rows;

      wrapper.style.width = vw+'px';
      grid.style.width    = vw+'px';
      grid.style.height   = vh+'px';
      grid.style.gridTemplateColumns = `repeat(${cols}, ${tileW}px)`;
      grid.style.gridTemplateRows    = `repeat(${rows}, ${tileH}px)`;

      tiles.forEach((t,i)=>{
        const c = i%cols, r = Math.floor(i/cols);
        t.style.width              = tileW+'px';
        t.style.height             = tileH+'px';
        t.style.backgroundImage    = `url(${urlImagen})`;
        t.style.backgroundSize     = `${vw}px ${vh}px`;
        t.style.backgroundPosition = `-${c*tileW}px -${r*tileH}px`;
      });

      const rect = wrapper.getBoundingClientRect();
      startY     = rect.top + window.pageYOffset;
      revealEndY = startY + vh;
      holdEndY   = revealEndY + vh;

  function mostrarPopup(el, texto) {
    document.querySelectorAll('.popup-box').forEach(p => p.remove());
    const popup = document.createElement('div');
    popup.className = 'popup-box';
    popup.textContent = texto;
    document.body.appendChild(popup);
    const rect = el.getBoundingClientRect();
    popup.style.left = `${rect.left + window.scrollX}px`;
    popup.style.top = `${rect.top + window.scrollY - 40}px`;
    popup.style.display = 'block';

    function cerrarPopup(event) {
      if (!popup.contains(event.target) && event.target !== el) {
        popup.remove();
        document.removeEventListener('click', cerrarPopup);
      }
    }
    setTimeout(() => document.addEventListener('click', cerrarPopup), 10);
  }
